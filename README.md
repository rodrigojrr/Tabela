PaintGrid
Uma ferramenta interativa para criar e colorir matrizes NxN no navegador.
## 🔗 Acesse o Projeto
**GitHub Pages:** https://rodrigojrr.github.io/paintgrid/
## 🎨 Funcionalidades
- **Criação Dinâmica:** Gere matrizes de qualquer tamanho (NxN)
- **Sistema de Cores:** Escolha entre 5 cores para pintar as células
- **Persistência:** Seu trabalho é salvo automaticamente no navegador
- **Dark Mode:** Interface moderna com tema escuro
## 🛠️ Tecnologias
- HTML5
- CSS3
- JavaScript (Vanilla)
## 💡 Como Usar
1. Digite o número de linhas/colunas no campo indicado
2. Clique em "Criar Matriz"
3. Selecione uma cor e clique nas células para pintá-las
4. Seus dados são salvos automaticamente!
